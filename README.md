Descripción del Módulo:

Spiderfoot es una herramienta muy usada para OSINT que automatiza la recopilación de información a través de diferentes recursos como fuentes públicas o herramientas privadas. 

Esta herramienta, que cuenta con licencia libre, está desarrollada en Python y sigue una estructura de clases. Su versión gratuita cuenta con más de 200 módulos categorizados según la información que extraen o el dato que solicitan.

El nuevo módulo para la herramienta de Spiderfoot,  sfp_puertos_abiertos.py  analiza el dominio que recibe, averigua su IP e indica sus Puertos Abiertos.
